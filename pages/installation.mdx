# Installation et configuration de l'environnement de développement

## Installation de Node.js et npm

Node.js est une plateforme logicielle qui vous permet d'exécuter du code JavaScript côté serveur. npm est le gestionnaire de paquets par défaut pour Node.js.

Pour installer Node.js et npm, vous pouvez télécharger et installer Node.js depuis le [site officiel](https://nodejs.org/). L'installation de Node.js installera également npm.

## Création d'un nouveau projet Angular avec Angular CLI

Angular CLI est un outil de ligne de commande qui vous permet de générer et de maintenir des projets Angular. Pour l'installer, ouvrez votre terminal et exécutez la commande suivante :

```bash copy
npm install -g @angular/cli
```

Une fois Angular CLI installé, vous pouvez créer un nouveau projet Angular en exécutant la commande suivante :

```bash copy
ng new my-app
```

Remplacez `my-app` par le nom de votre choix pour votre projet.

## Introduction à l'éditeur de code Visual Studio Code

Visual Studio Code est un éditeur de code source léger mais puissant qui fonctionne sur votre bureau. Il est disponible pour Windows, macOS et Linux. Vous pouvez le télécharger depuis le [site officiel](https://code.visualstudio.com/).

Après avoir installé Visual Studio Code, vous pouvez ouvrir votre projet Angular en exécutant la commande suivante dans votre terminal :

```bash copy
cd my-app
code .
```

Remplacez `my-app` par le nom de votre projet.

Maintenant, vous avez configuré votre environnement de développement pour Angular. Vous êtes prêt à commencer à construire votre première application Angular !
