# Directives Structurelles Angular

Les directives structurelles sont des outils puissants dans Angular qui vous permettent de modifier la structure du DOM de manière dynamique. Elles peuvent ajouter, supprimer et manipuler des éléments HTML directement dans vos templates. Dans ce chapitre, nous allons explorer les directives structurelles les plus couramment utilisées dans Angular, telles que `*ngFor`, `*ngIf`, `*ngSwitch`, et d'autres. Vous apprendrez comment elles peuvent être utilisées pour contrôler l'affichage du contenu, gérer des listes de données, et réaliser des opérations conditionnelles dans vos applications Angular.

## Objectifs d'apprentissage

- Comprendre le rôle et le fonctionnement des directives structurelles dans Angular.
- Apprendre à utiliser `*ngFor` pour afficher des listes de données dynamiquement.
- Découvrir comment `*ngIf` et l'attribut `*else`peuvent être utilisés pour afficher du contenu conditionnellement.
- Explorer l'utilisation de `*ngSwitch` pour une logique de sélection multiple.
- Utiliser `*ngContainer` pour grouper des éléments sans ajouter de nœuds supplémentaires au DOM.

À la fin de ce chapitre, vous serez capable de manipuler efficacement la structure de vos templates Angular en utilisant ces directives puissantes, ce qui vous permettra de créer des interfaces utilisateur dynamiques et réactives.

Préparez-vous à plonger dans le monde des directives structurelles et à découvrir comment elles peuvent enrichir vos applications Angular !
